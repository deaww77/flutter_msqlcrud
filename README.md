# Flutter + PHP + MySQL CRUD App

## Features

- สมัครสมาชิก
- ล็อกอิน
- แสดงรายชื่อผู้ใช้
- แก้ไข / ลบผู้ใช้

## Backend Setup

1. Import MySQL จากไฟล์ `schema.sql`
2. ตั้งค่า `api/db.php` โดย copy จาก `db.example.php`

## Flutter Setup

```bash
cd flutter_app
flutter pub add http
flutter pub get
flutter run
```
